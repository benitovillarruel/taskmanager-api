# taskmanager-api

Este es un programa básico en Python que permite sumar dos números ingresados por el usuario.

## 📌 Descripción

El programa:

1. Solicita al usuario dos números.
2. Realiza la suma de ambos valores.
3. Muestra el resultado en pantalla.

## 🖥️ Código

```python
# Programa para sumar dos números

# Pedimos al usuario que ingrese el primer número
numero1 = float(input("Ingresa el primer número: "))

# Pedimos al usuario que ingrese el segundo número
numero2 = float(input("Ingresa el segundo número: "))

# Calculamos la suma
suma = numero1 + numero2

# Mostramos el resultado
print("La suma de", numero1, "y", numero2, "es:", suma)
```

## ▶️ Instalaciòn: Cómo ejecutar el programa

1. Asegúrate de tener Python instalado.
2. Guarda el código en un archivo llamado `suma.py`.
3. Abre una terminal o consola.
4. Ejecuta el siguiente comando:

```bash
python suma.py
```

## 📋 USO: Ejemplo de ejecución

```bash
Ingresa el primer número: 5
Ingresa el segundo número: 7
La suma de 5.0 y 7.0 es: 12.0
```
